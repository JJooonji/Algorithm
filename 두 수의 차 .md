## 문제설명
#### 정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

<br>

## 제한사항
* -50000 ≤ num1 ≤ 50000
* -50000 ≤ num2 ≤ 50000

<br>

## 기본코드
```
function solution(num1, num2) {
    var answer = 0;
    return answer;
}
```


## 문제풀이
```
function solution(num1, num2) {
    var answer = 0;
    answer = num1 - num2
    return answer;
}
```
