<h2>문제풀이


    function solution(num) {
        var answer = ''본
        if(num % 2 === 0){
            answer = "Even"
        }else {
            answer = "Odd"
        }
        return answer;
    }
1. num을 2로 나눴을 때 0이 되면 짝수<br>
2. 만약에 짝수라면 answer이 Even라고 출력<br>
3. 짝수가 아니라면 홀수->answer에 Odd라고 출력
